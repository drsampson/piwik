Piwik # open source web analytics

piwik is the new version of phpMyVisites.

piwik is the future of open source web analytics: plugins based, all the data accessible via APIs (in XML, JSON, CSV, PHP), ajaxified interface.

piwik is still under heavy development and we are welcoming your help!

If you are a developer, or want to help with translations or with testing, feel free to join the party (tell us by email!).

you can try the online demo that shows you the latest piwik release.

Using Piwik

1 - download the last version (built every hour from the svn trunk/)

      or download the SVN trunk development version: 

   #svn co https://piwik.svn.sourceforge.net/svnroot/piwik/trunk

2 - then read the /README for more information about the installation (it is so easy!) and getting your first statistics!
